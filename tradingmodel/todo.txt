  # 1. 이동평균 교차 전략 (Moving Average Crossover Strategy)
  #    - 단기 이동평균선과 장기 이동평균선을 비교하여 매수 및 매도 신호를 생성하는 전략
  #    - 매수: 단기 이동평균선이 장기 이동평균선을 상향 돌파할 때
  #    - 매도: 단기 이동평균선이 장기 이동평균선을 하향 돌파할 때
  
  # 2. 모멘텀 전략 (Momentum Strategy)
  #    - 주가의 변화량을 추적하여 상승 모멘텀이 있는 경우 매수, 하락 모멘텀이 있는 경우 매도
  #    - 매수: 최근 주가가 이전보다 상승할 때
  #    - 매도: 최근 주가가 이전보다 하락할 때
  
  # 3. 다이버전스 전략 (Divergence Strategy)
  #    - 가격 차트와 기술적 지표(예: 이동평균선) 사이의 불일치를 활용하는 전략
  #    - 매수: 가격이 하락하는데 지표가 상승할 때
  #    - 매도: 가격이 상승하는데 지표가 하락할 때
  
  # 4. 트렌드 팔로잉 전략 (Trend Following Strategy)
  #    - 가격의 일정한 추세를 따라가는 전략으로 상승세일 때 매수, 하락세일 때 매도
  #    - 매수: 일정 기간 동안 주가가 상승세일 때
  #    - 매도: 일정 기간 동안 주가가 하락세일 때
  
  # 5. MACD 전략 (Moving Average Convergence Divergence Strategy)
  #    - MACD 지표와 신호선의 교차를 이용하여 추세 변화를 감지하는 전략
  #    - 매수: MACD가 신호선을 상향 돌파할 때
  #    - 매도: MACD가 신호선을 하향 돌파할 때
  
  # 6. 차트 패턴 분석 (Chart Pattern Analysis)
  #    - 차트 패턴(예: 삼각형, 쐐기형)을 분석하여 매수 및 매도 신호를 생성하는 전략
  #    - 매수/매도: 특정 패턴의 돌파 시점에서 신호를 생성
  
  # 7. 바이 앤 홀드 전략 (Buy and Hold Strategy)
  #    - 주식을 매수한 후 장기적으로 보유하여 수익을 내는 전략
  #    - 중간에 매수, 매도를 반복하지 않고 보유 상태를 유지함
  #    - 매수: 초기에 매수 후 계속 보유
  #    - 수익률 계산: 매수 후 일정 기간 후의 주가 변화에 따른 수익률 비교
